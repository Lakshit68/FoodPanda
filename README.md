## FoodTown Zomato-like MERN App

This is a Zomato-style food discovery and ordering app built with the MERN stack.

### Features
- User auth (register/login/logout) with httpOnly JWT cookies
- Browse restaurants, search, filter, and paginate
- Restaurant page with menus, ratings, and reviews
- Cart and checkout (mock payment)
- Order history

### Tech
- Backend: Node.js, Express, MongoDB (Mongoose)
- Frontend: React (Vite), React Router, Axios

### Getting Started
1) Backend
```
cd backend
cp .env.example .env
npm install
npm run dev
```

2) Frontend
```
cd frontend
npm install
npm run dev
```

Open http://localhost:5173

### .env (backend)
See `.env.example` in `backend/`.

### Notes
This project includes minimal, extensible implementations to get you started quickly.
